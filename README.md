# Digenic & Oligogenic Disease Literature Viewer

An interactive web viewer for exploring research papers on digenic and oligogenic disease inheritance patterns.

## 🌐 Live Website

**https://slavebot007-star.github.io/digenic-literature-viewer/**

## Features

- 🔍 **Search**: Find papers by title, authors, journal, or keywords
- 📅 **Filter by Year**: Focus on specific time periods (2019-2026)
- 📊 **Categories**: Browse by Clinical, Tools, or Review papers
- 📋 **Copy Citations**: One-click citation copying
- 📱 **Responsive**: Works on desktop and mobile
- 🌙 **Dark Theme**: Easy on the eyes
- 📥 **Export**: Download data as JSON

## Data

- **Total Papers**: 1,238 research papers
- **Year Range**: 2019 - 2026
- **Categories**:
  - Clinical: 1,161 papers (93.8%)
  - Tools: 51 papers (4.1%)
  - Review: 26 papers (2.1%)

## Sources

Comprehensive search from multiple academic databases:
- **PubMed**: 953 papers
- **Europe PMC**: 227 papers (includes bioRxiv/medRxiv preprints)
- **OpenAlex**: 200 papers
- **Crossref**: 100 papers

## Methodology

1. **Search**: Multi-source literature search using standardized queries
2. **Deduplication**: Advanced deduplication using DOI, PMID, and title similarity
3. **Categorization**: ML-based classification using MeSH terms, keywords, and journal metadata
4. **Validation**: Manual review of categorization results

Visit the live website or open `index.html` in any modern web browser.
