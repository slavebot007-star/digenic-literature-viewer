# Digenic & Oligogenic Disease Literature Viewer

An interactive web viewer for exploring research papers on digenic and oligogenic disease inheritance patterns.

## Features

- 🔍 **Search**: Find papers by title, authors, journal, or key findings
- 📅 **Filter by Decade**: Focus on specific time periods
- 📊 **Sort**: Newest or oldest papers first
- 📱 **Responsive**: Works on desktop and mobile
- 🌙 **Dark Theme**: Easy on the eyes
- 📥 **Export**: Download data as JSON

## Data

- **Total Papers**: 45+ research papers
- **Year Range**: 1994 - 2026
- **Sources**: PubMed, Semantic Scholar, OpenAlex, Crossref

## Usage

Open `index.html` in any modern web browser, or visit the GitHub Pages deployment.

## Deployment

This project is configured for GitHub Pages deployment from the `/docs` folder.
